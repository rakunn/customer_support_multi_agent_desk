# Troubleshooting Uploads

## TECH-CSV-UPLOAD: CSV Upload Failures

For CSV upload crashes, ask for the file size, browser, error text, and whether the CSV includes unsupported columns. Suggest retrying with a smaller file and the latest template.

## TECH-BROWSER-CACHE: Browser Cache

When a page fails repeatedly, ask the customer to refresh, clear cache for the site, and retry in a private browser window.
